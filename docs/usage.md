# Usage Guide

Learn how to use YouTabs effectively for tab management and page searching.

## Getting Started

### Opening YouTabs

1. **Sidebar** - Right-click toolbar → "YouTabs" → "Open Sidebar"
2. **Popup** - Click the YouTabs icon in the toolbar

### Basic Interface

The interface consists of:

- **Tab List** - Shows all open tabs
- **Group Panel** - Manage tab groups
- **Search Bar** - Search indexed pages
- **Settings** - Configure extension

## Tab Management

### Creating a New Group

1. Click the "+" button in the groups panel
2. Enter a name for your group
3. (Optional) Select a color
4. Click "Create"

### Adding Tabs to Groups

**Method 1: Drag and Drop**
- Drag a tab onto a group
- The tab will be added to that group

**Method 2: Context Menu**
- Right-click a tab
- Select "Move to Group" → Choose group

### Organizing Tabs

- **Reorder Tabs** - Drag tabs within the list
- **Move Between Groups** - Drag to different group
- **Remove from Group** - Drag to "Ungrouped"

### Group Controls

Each group has controls for:

- **Collapse/Expand** - Click the arrow icon
- **Rename** - Double-click group name
- **Change Color** - Click color picker
- **Delete** - Right-click → Delete

## Using the Search

### Performing a Search

1. Click the search bar or press `Ctrl+F`
2. Type your search query
3. Results appear instantly
4. Click a result to navigate

### Search Filters

Filter results by content type:

- **All** - Search everything
- **Headings** - Only headings
- **Paragraphs** - Only text content
- **Links** - Only hyperlinks
- **Images** - Only images

### Search Tips

- Use specific terms for better results
- Search for page titles you remember
- Use headings to find specific sections

## Page Indexing

### Automatic Indexing

Pages are indexed automatically when you visit them. The extension extracts:

- Page title
- All headings (H1-H6)
- Paragraph text
- Link text and URLs
- Image alt text and sources
- Div elements
- UL/OL Lists and list items
- Input form elements
- Video and audio elements
- IFrame embeddings
- Span text elements

### Manual Indexing

Add custom text to the index:

1. Select text on any webpage
2. Right-click → "Add to YouTabs Index"
3. The text is now searchable

### Viewing Indexed Pages

Access your indexed content:

1. Open the sidebar
2. Navigate to "Indexed Pages"
3. View all indexed websites

## Sound Management

### Muting Tabs

1. Find a tab with audio (indicated by speaker icon)
2. Click the mute button
3. Tab is now muted

### Unmuting Tabs

1. Click the muted icon on the tab
2. Audio is restored

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+F` | Focus search |
| `Escape` | Clear search |

## Settings Configuration

### Accessing Settings

1. Click the gear icon in the interface
2. Browse setting categories
3. Adjust as needed

### Available Settings

- **Grouping Mode** - How tabs are grouped
- **Group Behavior** - Collapse/expand defaults
- **Search Preferences** - Default filters
- **Keyboard Shortcuts** - View/change shortcuts

## Tips & Tricks

### Efficient Tab Management

1. **Use Colors** - Color-code groups by project
2. **Regular Cleanup** - Close unused tabs weekly
3. **Name Groups Clearly** - Use descriptive names

### Better Searches

1. **Index Important Pages** - Manually add key information
2. **Use Headings** - Navigate to specific sections
3. **Filter Results** - Use filters for precision

### Workflow Integration

1. **Daily Workflow** - Create groups for daily tasks
2. **Project Groups** - One group per project
3. **Quick Access** - Pin important groups

## Troubleshooting

### Search Not Working

- Ensure pages have been indexed
- Check your search query
- Verify filter settings

### Groups Not Saving

- Check browser storage
- Ensure no conflicts with other extensions
- Try recreating the group

### Performance Issues

- Reduce number of open tabs
- Limit indexed pages
- Clear old indexed data
